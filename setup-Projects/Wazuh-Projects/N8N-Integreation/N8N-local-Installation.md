# N8N Installation in Ubuntu

## Step 1: Install Node.js and npm
Using Terminal:
```bash
sudo apt install nodejs
sudo apt install npm
```
## Step 2: Install n8n Globally
Using Terminal:
```bash
sudo npm install -g n8n
```
## Step 3: Start n8n
Using Terminal: `n8n`

**URL :** http://localhost:5678
